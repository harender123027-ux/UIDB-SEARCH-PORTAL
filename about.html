Pehchan by Haryana Police

Unified Biometric Identification System — State-of-the-Art Facial Recognition & Analytics

Functionality

  • AdaFace Integration: High-accuracy facial embeddings optimized for low-quality surveillance footage.
  • Quality Scoring: Real-time image quality assessment (L2 Norm) to ensure biometric reliability.
  • UI Body Search: Face, attribute and voice-note matching against unidentified-body submissions. (Criminal-database and missing-person matching are out of scope for Phase 1.)
  • Robust Fallback: Center-crop fallback ensures processing even when detection landmarks are missing.

Architecture

Cloud-native hybrid architecture designed for speed and state-level scale.

  • Decoupled Models: AI weights (~1GB) are mounted via Azure Blob Storage for ultra-fast deployments.
  • Vector Search: Qdrant-powered sub-millisecond similarity lookups for 100k+ records.
  • Micro-Background Tasks: Celery/Redis pipeline for asynchronous, non-blocking biometric processing.

Tech Stack

FastAPI (Python) React (PWA) Qdrant (Vector DB) PostgreSQL (Azure) PyTorch InsightFace AdaFace Docker & ACR

Modern, open-source AI stack with enterprise-grade cloud orchestration.
